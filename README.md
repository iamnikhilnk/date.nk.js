date.nk.js
==========

### Date formatting library

date.nk.js is a date formatting library which can convert the date time to string in a generic format.


# Syntax

```
var date = new Date();
console.log(date.format("MMM dd yyyy, hh:mm:ss:fff tt"));
```


# Available formats

"d"
"dd"
"fff"
"hh"
"HH"
"mm"
"M"
"MM"
"MMM"
"MMMM"
"ss"
"tt"
"y"
"yy"
"yyyy"
"w"
"www"
"wwww"
